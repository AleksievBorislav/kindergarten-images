# Kindergarten Images Repository

This folder structure is for organizing images for the kindergarten website.

## Folders

- **gallery/** - Images for the gallery section
- **staff/** - Photos of staff members

## To Use These Images

1. Add your image files to the appropriate folders
2. Push to GitHub
3. Use raw URLs in JSON files like: `https://raw.githubusercontent.com/AleksievBorislav/kindergarten-images/main/gallery/image-name.jpg`

## Example Structure

```
gallery/
  - playground.jpg
  - art-activity.jpg
  - celebration.jpg

staff/
  - teacher-1.jpg
  - teacher-2.jpg
  - assistant-1.jpg
```
